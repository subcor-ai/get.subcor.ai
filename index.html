Subcor CLI

Author and validate system models, and manage systems, releases, instances, and collectors — from your terminal.

$ curl -fsSL https://get.subcor.ai/install.sh | sh

Installs to ~/.subcor/bin on macOS and Linux (amd64/arm64) — no sudo required.

$ brew install subcor-ai/tap/subcor

macOS only. Upgrade with brew upgrade subcor, remove with brew uninstall subcor.

For more details and options: https://github.com/subcor-ai/get.subcor.ai
